Minetest Theme
for Twitter Bootstrap

Free theme to use on your own pages.

Download GitHub Minetest CoRNeRNoTe

Features

  • Offers three sections for the user to dig (scroll) through.
  • Easily replace the main image and it will show transparently through the grass.
  • Basic craft table styles.
  • Popup screenshots with Facebox2.
  • Code syntax highlighting with Prettify.
  • Built using the sleek, intuitive, and powerful Twitter Bootstrap.
  • Does not require the above css/js libraries to be downloaded locally thanks to cdnjs.
